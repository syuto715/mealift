## App Store 向け本文 (このまま貼り付け可能)

```
v1.5 では「ミー先生」 が新登場、 食事 / トレーニングを 1 人で続ける
ための専属コーチがあなたに寄り添います。

【新機能】
・AI コーチ「ミー先生」 とのチャット — 食事 / トレーニングの質問
  に応答が token-by-token で表示。 落ち着いた口調 / 過度な賞賛なし
  / サイエンスベース。
・診断ウィザード — 7 つの質問にお答えいただくと、 経験 / 目標 / 機材
  / 制限から最適なトレーニングルーティンを生成。
・ルーティン自動生成 — 「自宅で肩と背中の日」 のような自然な文章で
  意図を伝えると、 ルーティンを設計しメニューに追加できます。
・週次アドバイス — 週ごとの振り返り + 今週の重点 + 1 行 CTA が届きます。
・食事ログのスピード入力 (Quick log) — 1 タップで現在のミールに追加。
・お気に入り食品 (★) — 検索結果の ★ ボタンでお気に入り保存、 次回から
  上位に表示。
・最近よく使った食品の上位表示 — 自然な使用頻度順に並び替え。
・料理画像からの栄養推定 (Vision、 Pro プラン) — 撮影した料理から
  料理名と材料を推定。

【改善】
・アクセシビリティ強化 — 全 4 つのコーチ surface で VoiceOver 対応を
  強化。 ボタンの状態 (送信不可の理由 / 破棄前の確認 等) が読み上げに
  反映されます。
・ペルソナ表記の統一 — 「ミー先生」 を大文字、 「AI コーチ」 を
  sub-label として全 surface で一貫表示。
・会話アーカイブ / 削除 — 会話一覧で長押し → アーカイブ (隠す) /
  削除 (元に戻せない) を選択可能。 削除は 2 段階確認。
・パフォーマンスの最適化 と 細かなバグ修正。
```
